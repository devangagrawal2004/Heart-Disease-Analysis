# ❤️ Heart Disease Analysis Dashboard

## Project Overview

This project was developed during the NASSCOM Data Analytics with Tableau Internship.

The objective of this project is to analyze heart disease data using Tableau and generate meaningful insights through interactive dashboards and stories. The project supports doctors, government authorities, and individuals in understanding the major factors associated with heart disease.

---

## Objectives

- Analyze heart disease dataset.
- Identify important health risk factors.
- Build interactive dashboards using Tableau.
- Support healthcare decision-making.
- Present insights using Tableau Story.

---
